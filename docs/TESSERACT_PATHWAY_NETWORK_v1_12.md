# Tesseract Pathway Network v1.12

v1.12 adds the Stairway Compression Governor.

## Purpose

The system can now compress safe adjacent control layers into one governed assessment:

```text
performance probe
→ drift/regression judgment
→ proposal receipts
→ approval requirement
→ compressed next-bundle recommendation
```

## What can be compressed

The safe compressed bundle is:

```text
drift_regression_sentinel
+ proposal_receipt_engine
+ human_approval_gate
```

These can be bundled because they observe, score, and require approval. They do not apply code changes.

## Run

```powershell
.\scripts\run_tesseract_stairway.ps1
```

## Boundary

This is roadmap compression and proposal governance only. It does not mutate code, apply patches, run continuously, or claim AGI.
