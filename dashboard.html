Conclusions & Emergent Ideas
📋
Test Conclusions

What We Measured

  • DataLearner Regression R²=0.982 — The RegressionNet (128→64→32) with BatchNorm and Dropout learns the linear relationship 3x₁+2x₂ with near-perfect accuracy. 1.81s training on 200 samples.
  • Classification 98.67% — 3-class 2D blobs are cleanly separable. The ClassificationNet achieves near-perfect accuracy in 0.52s.
  • Forecasting R²=0.225 — Sine wave extrapolation is inherently difficult for LSTMs trained on single-step prediction. The model learns the local pattern but struggles with phase alignment for future steps.
  • Pattern Engine Seasonal r=0.78 — The 1D-CNN SeasonalPredictor captures repeating patterns well. The sine wave is the ideal case.
  • Pattern Detector Over-classifies as Seasonal — 5 of 6 patterns detected as "seasonal" because the autocorrelation heuristic picks up mean-reversion and trends as cycles. Needs calibration.
  • Quality Predictor Memory r=0.97 — Memory usage is the most predictable from training dynamics (parameter count dominates). Accuracy r=0.84, Latency r=0.54 with only 4 histories.
  • Full Pipeline NaN Loss — The 1.56M parameter CNN is too large for random 500-sample data. Learning rate needs tuning or model capacity reduction.
